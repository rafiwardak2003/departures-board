# Departures Board

This application requires the backend be ran along with the 
client as the client fetches the backend for data.

## To Run Both the Server and Client
`cd client/`

`npm install`

`cd ../`

`npm install`

`npm start`

`http://localhost:3000`

## To Run Only the Server
`npm install`

`npm run server`

`http://localhost:3001`

## To Run Only the Client
`cd client/`

`npm install`

`npm start`

`http://localhost:3000`

## To Test
`cd client/`

`npm install`

`npm test`

## To Build for Production
`cd client/`

`npm install`

`npm run build`

`navigate to /build`
